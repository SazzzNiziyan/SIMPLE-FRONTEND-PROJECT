# Arik — Modern Portfolio Template

A sleek, minimal, and modern portfolio template to showcase your work, skills, and personality. Arik is designed to be lightweight, responsive, and easy to customize so you can get your portfolio online fast.

---

## Features

- Clean, modern design with attention to typography and spacing  
- Fully responsive — looks great on mobile, tablet, and desktop  
- Sections for Hero, About, Projects, Skills, Testimonials, Blog, and Contact  
- Smooth scroll and subtle animations to enhance UX  
- Easy to customize colors, fonts, and content  
- Ready for static hosting (GitHub Pages, Netlify, Vercel, Surge, etc.)

---

## Built with

- HTML5
- CSS3 (plain CSS / easily replaceable with SCSS or Tailwind)
- JavaScript (vanilla; can be adapted to frameworks like React, Vue, or Svelte)

## Quick start

Clone the repo and open index.html:

```bash
# Clone
git clone https://github.com/SazzzNiziyan/Arik---Modern-Portfolio-Template.git

# Open the project (double-click index.html or serve with a static server)
# Example using a simple Python http server:
cd Arik---Modern-Portfolio-Template
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

If you use Node.js and want a dev server with live reload, install a simple server:

```bash
npm install -g live-server
live-server
```

## Customization

Make the template yours in a few simple steps:

- Replace content in `index.html` (or the relevant HTML files) to update text and section order.  
- Edit CSS variables at the top of your stylesheet to change color scheme and spacing.  
- Swap fonts by updating the Google Fonts link in the head.  
- Replace image assets in the `assets/` folder (optimize images for web).  
- For contact forms, connect to your preferred backend or use services like Formspree, Netlify Forms, or EmailJS.

Tips:
- Keep your hero concise — a short tagline with a clear call-to-action converts better.
- Show 3–8 of your best projects with short descriptions and links to case studies or repos.
- Add a downloadable resume link for recruiters.

## Deployment

Deploy to GitHub Pages:
1. Push your changes to the `main` branch.
2. Go to repository Settings → Pages → Build and Deployment → Set source to `main` (root).
3. Your site will be live at `https://<username>.github.io/<repo>/`.

Or use Netlify / Vercel:
- Connect your repository and choose the build settings (for static sites no build command needed).
